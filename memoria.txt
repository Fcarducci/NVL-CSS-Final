En el presente proyecto se ha creado una plantilla tipo de un blog, se busco por internet una plantilla modelo ya diseñada y se aplicaron los conocimientos adquiridos en los modulos (HTML y CSS) para lograr o asemejar el diseño de la plantilla buscada, primeramente se creo toda la estructura HTML, se aplicaron etiquetas nuevas como "<header>, <article>, <footer>, <blockquote>" para darle mas informacion a la web, una vez definida toda la estructura HTML, se creo un nuevo archivo llamado "Styles.css" en donde se ancló este archivo en el HTML a traves de la etiqueta "<link>" dentro del "<head>", posteriormente se aplicaron multiples propiedades CSS vistas en el temario del modulo y otras investigada por mi, se usaron propiedades como: "margin, border, padding, display: flex, display: inline-flex, font-size, background, width, text-align, etc.

Todas estas propiedades se usaron para modificar el estilo predeterminado que trae el HTML, se cambio el color de fondo, el tamaño de la letra de la primera columna, se uso display flex para colocar etiquetas una al lado de otras y no por debajo como es usualmente, tambien se aplicaron margins y padding asi como tambien border, todo esto con la finalidad de que se vea un diseño web lo mas profesional posible.

Finalmente ya diseñado el blog, se envio el archivo a Git Hub en la terminal del editor de codigo "Visual studio code", se aplico "git add .", "git commit" y "git push".
